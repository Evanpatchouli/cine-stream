# CineStream 权限系统说明

## 当前角色

- `SUPER_ADMIN`: 超级管理员，拥有全部管理权限。
- `CONTENT_ADMIN`: 内容管理员，可管理影视和剧集资源。
- `OPERATOR`: 运营人员，可进入管理端处理日常内容配置。
- `NORMAL_USER`: 普通客户端用户，可登录客户端观看影视。

## 当前权限范围

系统用户：

- 查看用户列表
- 添加用户
- 查看用户详情
- 编辑用户信息

影视管理：

- 查看影视列表
- 新增影视
- 编辑影视
- 删除影视
- 配置剧集
- 浏览服务器视频文件

客户端：

- 查看影视
- 播放影视剧集

## 同步规则

- 前端权限常量维护在 `admin/src/constants/permissions.ts`。
- 后端角色常量维护在 `server/src/constants/role.constant.ts`。
- 共享角色常量维护在 `common/src/constants/role.ts`。
- 初始角色和权限数据维护在 `server/src/migrator/v1.ts`。
- 前端权限只控制菜单和按钮展示，后端仍需要通过鉴权装饰器校验接口访问。
