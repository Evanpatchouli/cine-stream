# 当前任务

## 自定义全屏按钮交互修正

### 背景

用户反馈上一版不符合预期：

- 自定义全屏按钮遮住了原生速度按钮。
- 希望点击视频、进度等原生控件浮现时，全屏按钮才跟着展示。
- 希望全屏状态下该按钮变成退出全屏按钮。

### 方案

- 原生 fullscreen 控件改为透明隐藏并保留占位，避免速度按钮右移到全屏位。
- 自定义按钮贴到右下角原生 fullscreen 占位区域，去掉常驻圆形背景。
- 播放器交互时短暂显示按钮，无交互后隐藏。
- 监听 fullscreenchange，同一按钮在全屏状态下切换为退出全屏并调用 exitFullscreen。

### 状态

实施中。
